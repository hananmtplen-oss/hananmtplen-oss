## About Me

I am a security researcher specializing in web application security, bug bounty hunting, and responsible vulnerability disclosure. My work focuses on identifying and responsibly disclosing security vulnerabilities in web applications, APIs, and authentication systems.

### Core Competencies
- **Web Application Security**: OWASP Top 10, injection attacks, authentication bypasses
- **API Security**: REST API testing, GraphQL security, API key management
- **Authentication Systems**: JWT, OAuth, session management, MFA bypass testing
- **Client-Side Security**: XSS, CORS, open redirects, DOM-based vulnerabilities
- **Server-Side Security**: SSRF, path traversal, SSRF, command injection
- **Cryptography**: Weak implementations, insecure random number generation, padding oracle attacks

2. **Discovery**: Manual testing with Burp Suite, OWASP ZAP, and browser automation
3. **Exploitation**: Proof-of-concept development using Python, JavaScript, and Bash
4. **Reporting**: Detailed reports with impact assessment and remediation recommendations

## Disclosure Policy

I strictly follow responsible disclosure practices:
- Report vulnerabilities privately to vendors first
- Allow 90 days for patching before public disclosure
- Never exploit vulnerabilities for personal gain
- Provide detailed proof-of-concept and remediation guidance
